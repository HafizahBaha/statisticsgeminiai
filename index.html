📚 Key Statistics Words

Click each card to flip it and see the explanation and example.

Variable
Tap to see explanation
Anything you measure or record in your experiment.
🌱 e.g. plant height, treatment type, leaf number
Independent Variable (IV)
Tap to see explanation
The thing you change or control in your experiment.
🧪 e.g. fertilizer level (Control / Low / High)
Dependent Variable (DV)
Tap to see explanation
The thing you measure — it depends on what you did to the plants.
📏 e.g. plant height (cm), fresh weight (g)
Numerical Variable
Tap to see explanation
A variable measured as a number you can calculate with.
📊 e.g. height = 24.5 cm, weight = 8.3 g, leaves = 7
Categorical Variable
Tap to see explanation
A variable measured as a label or group — not a number.
🏷 e.g. Germinated / Not Germinated, Control / Fertilizer
p-value
Tap to see explanation
A number (between 0 and 1) that tells you if your result is real or just random chance.
📉 Rule: p < 0.05 = real result ✅
Significant Result
Tap to see explanation
When p < 0.05, the result is statistically significant — it is unlikely to be random chance.
✅ This means the fertilizer likely did make a difference!

📌 The Most Important Rule

p < 0.05
✅ Statistically SIGNIFICANT
The result is real — not just random chance
p ≥ 0.05
❌ NOT Significant
The difference might just be random chance

✅ Recommended Test: One-Way ANOVA
You have 3+ independent groups — ANOVA tests all at once to avoid false positives.
Does fertilizer level (Control, Low, High) affect plant height?
Fertilizer Level (3+ groups)
Plant_Height_cm (numerical)
F-statistic, df, p-value, post-hoc if significant

⚠
ANOVA tells you at least one group is different — but not which one. Run a post-hoc test (Tukey's HSD) to find out.

"A one-way ANOVA was conducted to compare [DV] across [groups]. There was a statistically significant effect of [IV] on [DV], F(df1, df2) = X.XX, p = 0.0XX."

Both variables are numerical — Pearson r measures how strongly they move together.

Pearson r, p-value

🧪 Learn the 4 Statistical Tests

Select a test below to explore when to use it, how to read the output, and how to write the result.

📊 Independent Samples t-test

Compare the mean of a numerical variable between exactly 2 independent groups.

✅ When to Use

Use when you are comparing the mean of a numerical variable between exactly 2 groups, and the groups are different (independent) plants.

Research Setup

Example QuestionIs plant height different between Control and High Fertilizer?
IVTreatment (2 groups)
DVPlant_Height_cm (numerical)
Outputt-statistic, df, p-value

Interpretation Guide

✅
p < 0.05: The two groups are significantly different.
❌
p ≥ 0.05: No significant difference between the groups.
📌
Focus on the p-value. The t-statistic shows the size of the difference.
✍ Result Sentence Template

"An independent samples t-test was conducted to compare plant height between the [Group A] and [Group B] groups. The result showed a statistically significant difference, t(df) = X.XX, p = 0.0XX. The [Group A] group had a higher mean plant height (M = X.XX cm) compared to [Group B] (M = X.XX cm)."

📈 One-Way ANOVA

Compare the mean of a numerical variable across 3 or more groups simultaneously.

✅ When to Use

Use when you are comparing the mean of a numerical variable across 3 or more groups.

⚠
Why not just run 3 t-tests? Running multiple t-tests increases your chance of a false positive. ANOVA tests all groups at once — it is the correct method.

Research Setup

Example QuestionDoes fertilizer level affect plant height across Control, Low, High?
IVFertilizer Level (3 groups)
DVPlant_Height_cm (numerical)
OutputF-statistic, df1, df2, p-value

Interpretation Guide

✅
p < 0.05: At least one group is significantly different.
❌
p ≥ 0.05: No significant difference across groups.
📌
ANOVA says someone is different. Run a Tukey's HSD post-hoc test to find out which pairs differ.
✍ Result Sentence Template

"A one-way ANOVA showed a statistically significant effect of fertilizer level on plant height, F(2, 57) = X.XX, p = 0.0XX. The High Fertilizer group had the highest mean plant height (M = X.XX cm), followed by Low Fertilizer (M = X.XX cm) and Control (M = X.XX cm)."

🔗 Pearson Correlation

Measure the strength and direction of the relationship between two numerical variables.

✅ When to Use

Use when you want to know if two numerical variables are related. You are NOT comparing groups — you are looking for a pattern.

⚠
Correlation is NOT causation! If r = 0.85 between plant height and fresh weight, it means they are related — not that height causes weight. Both might be caused by the fertilizer.

Understanding r (correlation coefficient)

r valueStrengthDirectionPlain English
+0.7 to +1.0StrongPositiveAs X goes up, Y clearly goes up too
+0.4 to +0.6ModeratePositiveSome upward trend
~0.0NoneNoneNo relationship at all
-0.4 to -0.6ModerateNegativeAs X goes up, Y tends to go down
-0.7 to -1.0StrongNegativeClear downward trend
✍ Result Sentence Template

"A Pearson correlation was conducted to examine the relationship between plant height and fresh weight. There was a statistically significant positive correlation between the two variables, r = 0.XX, p = 0.0XX. This indicates that as plant height increased, fresh weight also tended to increase."

🔲 Chi-Square Test of Independence

Test whether two categorical variables are statistically associated.

✅ When to Use

Use when both your variables are categorical (labels/groups) and you want to know if they are related to each other.

Research Setup

Example QuestionIs germination outcome linked to treatment group?
Variable 1Treatment (Control / Low / High)
Variable 2Germination (Germinated / Not Germinated)
Both TypesCategorical — labels, not numbers
Outputχ², degrees of freedom, p-value

Interpretation Guide

✅
p < 0.05: The two categorical variables ARE related.
❌
p ≥ 0.05: No significant association found.
📌
The contingency table shows the count of plants in each combination. Chi-square tests whether these counts are what you'd expect if there was no relationship.
✍ Result Sentence Template

"A chi-square test of independence was conducted to determine whether germination outcome was associated with treatment group. The result was statistically significant, χ²(2) = X.XX, p = 0.0XX, indicating that treatment group significantly influenced germination success."

💡
This compares plant height between Control and High Fertilizer groups. Look for the p-value — is it less than 0.05?
Cell 3 — Independent Samples t-test
# CELL 3 — Independent samples t-test
# Question: Is plant height different between Control and High Fertilizer?

group_control = df[df["Treatment"] == "Control"]["Plant_Height_cm"]
group_high    = df[df["Treatment"] == "High Fertilizer"]["Plant_Height_cm"]

print(f"Mean — Control       : {group_control.mean():.2f} cm")
print(f"Mean — High Fertilizer: {group_high.mean():.2f} cm")

t_stat, p_value = ttest_ind(group_control, group_high)
print(f"\nt-statistic : {t_stat:.4f}")
print(f"p-value     : {p_value:.4f}")

if p_value < 0.05:
    print("\n✅ SIGNIFICANT — There IS a difference between the two groups.")
else:
    print("\n❌ NOT SIGNIFICANT — There is NO clear difference.")
⚠
ANOVA tests all 3 groups at once. If p < 0.05, run a post-hoc test to find WHICH groups differ. Ask Gemini AI: "How do I run Tukey's HSD in Python?"
Cell 4 — One-Way ANOVA
# CELL 4 — One-Way ANOVA
# Question: Does fertilizer level affect plant height?

control   = df[df["Treatment"] == "Control"]["Plant_Height_cm"]
low_fert  = df[df["Treatment"] == "Low Fertilizer"]["Plant_Height_cm"]
high_fert = df[df["Treatment"] == "High Fertilizer"]["Plant_Height_cm"]

print(f"Mean — Control        : {control.mean():.2f} cm")
print(f"Mean — Low Fertilizer : {low_fert.mean():.2f} cm")
print(f"Mean — High Fertilizer: {high_fert.mean():.2f} cm")

f_stat, p_value = f_oneway(control, low_fert, high_fert)
print(f"\nF-statistic : {f_stat:.4f}")
print(f"p-value     : {p_value:.4f}")

if p_value < 0.05:
    print("\n✅ SIGNIFICANT — At least one group is different!")
    print("   Run a post-hoc test to find out WHICH groups differ.")
else:
    print("\n❌ NOT SIGNIFICANT — No clear difference across groups.")
🤖
This checks if taller plants also weigh more. Remember: correlation ≠ causation!
Cell 5 — Pearson Correlation
# CELL 5 — Pearson Correlation
# Question: Do taller plants also weigh more?

r, p_value = pearsonr(df["Plant_Height_cm"], df["Fresh_Weight_g"])

print(f"Pearson r : {r:.4f}")
print(f"p-value   : {p_value:.4f}")

if p_value < 0.05:
    direction = "positive" if r > 0 else "negative"
    strength  = "strong" if abs(r) > 0.7 else ("moderate" if abs(r) > 0.4 else "weak")
    print(f"\n✅ SIGNIFICANT — {strength} {direction} correlation.")
else:
    print("\n❌ NOT SIGNIFICANT — No clear relationship found.")
📌
The contingency table shows the COUNT of plants in each combination (e.g. how many Control plants germinated). Chi-square tests if these counts are what we would expect if there was NO relationship.
Cell 6 — Chi-Square Test
# CELL 6 — Chi-Square Test of Independence
# Question: Is germination outcome linked to treatment group?

ct = pd.crosstab(df["Treatment"], df["Germination"])
print("Contingency Table:")
print(ct)

chi2, p_value, dof, expected = chi2_contingency(ct)
print(f"\nChi-square value   : {chi2:.4f}")
print(f"Degrees of freedom : {dof}")
print(f"p-value            : {p_value:.4f}")

if p_value < 0.05:
    print("\n✅ SIGNIFICANT — Treatment and Germination ARE related!")
else:
    print("\n❌ NOT SIGNIFICANT — No significant association found.")

Run this cell after Cell 4 to see all 3 plots at once. Great for putting in your FYP!
